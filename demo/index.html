How It Works

Integration Steps

  1. Load Prebid.js - Include the script in your page
  2. Configure S2S - Point to our PBS endpoint
  3. Define Ad Units - Specify your ad placements
  4. Request Bids - Let Prebid.js handle the auction
  5. Render Ads - Display winning bids or passback to GAM
